Slo-Fashion

Slo-Fashion is designed as a personal wardrobe inventory that encourages and facilitates clothing recycling. Users can easily upload pictures of garments they wish to recycle. Our service takes the hassle out of sustainability by picking up the items directly from the user's location and delivering them to fashion brands committed to sustainability. In return for their contribution, users are rewarded with cryptocurrency, which can be used within our marketplace to purchase outfits from approved sustainable brands. This system not only incentivizes recycling but also promotes the circulation of sustainable garments in the fashion ecosystem.

Tech Stack

  • Languages: Python, JavaScript
  • Frontend: React.js for building dynamic and interactive user interfaces, Tailwind CSS for fast, utility-first styling.
  • Backend: Python for logic and data processing, Node.js for dependency management and tooling.
  • Authentication: Auth0 for secure, scalable user authentication.
  • Database: MongoDB for flexible, high-performance NoSQL data storage.C
  • Integrations:VerbWire for automated logistics, supporting effortless wardrobe management and recycling practices.
